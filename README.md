Здравствуйте!
Я завершил выполнение задания. Ниже приведены инструкции по запуску проекта

Установите зависимости
composer install

Скопируйте пример файла .env и создайте новый:
cp .env.example .env

Сгенерируйте ключ приложения
php artisan key:generate


Выполните миграции и сидеры
(Сидеры добавлены для упрощения тестирования данных)
php artisan migrate --seed

Запустите локальный сервер
php artisan serve
npm run dev
